# NuGet Publishing Guide (Archived)

> **Note:** ExcelMcp no longer publishes to NuGet. This document is kept for historical reference only.
>
> As of the standalone exe release, ExcelMcp distributes self-contained executables via GitHub Releases:
> - `ExcelMcp-MCP-Server-{version}-windows.zip` → `mcp-excel.exe`
> - `ExcelMcp-CLI-{version}-windows.zip` → `excelcli.exe`
>
> See [RELEASE-STRATEGY.md](RELEASE-STRATEGY.md) for the current release process.
> See [INSTALLATION.md](INSTALLATION.md) for installation instructions.
